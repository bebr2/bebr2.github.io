强化学习基础

做一些强化学习的记录，以备查看，后续可能会持续更新。

基本概念

随机变量通常用大写，观测到的常量用小写。

  1. Agent：可以理解为智能体。
  2. State：环境的某个状态。
  3. Action：Agent在某个时刻的选择。
  4. Policy：记作π，定义Policy Function：π(a∣s)=P(A=a∣S=s)\pi(a|s)=P(A=a|S=s)π(a∣s)=P(A=a∣S=s)。
  5. Reward：Agent做出Action后，环境返回的奖励。RiR_iRi​取决于SiS_iSi​和AiA_iAi​。
  6. State Transition：Agent做出Action后导致环境的State改变。可以是随机化的，如p(s′∣s,a)=P(S′=s∣S=s,A=a)p(s^{'}|s,a)=P(S^{'}=s|S=s,A=a)p(s′∣s,a)=P(S′=s∣S=s,A=a)。
  7. 两个随机变量：A∼π(⋅∣s),S′∼p(⋅∣s,a)A\sim \pi(·|s), S^{'}\sim p(·|s,a)A∼π(⋅∣s),S′∼p(⋅∣s,a)
  8. (state, action, reward) Trajectory：轨迹，指一次迭代（Episode）结束后，将(s, a, r)串起来：

  1. Discounted Return(cumulative discounted future reward)：折扣回报，定义为Ut=∑k=tγk−tRkU_t=\sum_{k=t}\gamma^{k-t}R_kUt​=∑k=t​γk−tRk​，上标通常是终止时刻n。这是一个随机变量，我们能观察到的是ut=∑k=tγk−trku_t=\sum_{k=t}\gamma^{k-t}r_kut​=∑k=t​γk−trk​。

    根据上式，可以推出：Ut=Rt+γUt+1U_t=R_t+\gamma U_{t+1}Ut​=Rt​+γUt+1​。

  2. Action-Value Function：动作价值函数，定义为Qπ(st,at)=E[Ut∣St=st,At=at]Q_\pi(s_t,a_t)=E[U_t|S_t=s_t,A_t=a_t]Qπ​(st​,at​)=E[Ut​∣St​=st​,At​=at​]，取决于st,at,πs_t,a_t,\pist​,at​,π 和 ppp。

  3. State-Value Function：状态价值函数，定义为Vπ(st)=EA[Qπ(st,A)]V_\pi(s_t)=E_A[Q_\pi(s_t,A)]Vπ​(st​)=EA​[Qπ​(st​,A)]，其中A∼π(⋅∣s)A\sim \pi(·|s)A∼π(⋅∣s)。评价状态s的好坏。

  4. Optimal action-value function：最优动作价值函数，定义为 Q∗(st,at)=maxπ[Qπ(st,at)]Q^*(s_t,a_t)=max_\pi[Q_\pi(s_t,a_t)]Q∗(st​,at​)=maxπ​[Qπ​(st​,at​)]。可以根据这个函数来选择下一个Action，即在状态s下，a∗=argmaxaQ∗(s,a)a^*=argmax_a Q^*(s,a)a∗=argmaxa​Q∗(s,a)。

  5. Optimal state-value function：最优状态价值函数，定义为V∗(s)=maxπVπ(s)V^*(s)=max_\pi V_\pi(s)V∗(s)=maxπ​Vπ​(s)，有V∗(s)=maxaQ∗(s,a)V^*(s)=max_aQ^*(s,a)V∗(s)=maxa​Q∗(s,a)（这个结论似乎有问题，因为maxaQ∗(s,a)=maxa,πQπ(s,a)max_aQ^*(s,a)=max_{a,\pi}Q_\pi(s,a)maxa​Q∗(s,a)=maxa,π​Qπ​(s,a)，怎么看都和左式不一样。。）。

  6. Optimal advantage function：最优优势函数，定义为A∗(s,a)=Q∗(s,a)−V∗(s)A^*(s,a)=Q^*(s,a)-V^*(s)A∗(s,a)=Q∗(s,a)−V∗(s)，可以理解为判断一个动作的好坏。

  7. DQN(Deep Q-Network)：希望学习Q∗(st,at)Q^*(s_t,a_t)Q∗(st​,at​)，使用神经网络Q(s,a;w)Q(s,a;w)Q(s,a;w)来近似它。

  8. Transition：指的就是一条记录(st,at,rt,st+1)(s_t,a_t,r_t,s_{t+1})(st​,at​,rt​,st+1​)

  9. Offline RL：离线强化学习，又称Batch RL，要求Agent只能从一个固定的数据集中学习而不能试错。

Actor就是策略网络π(a∣s;θ)\pi(a|s;\theta )π(a∣s;θ)，Critic就是价值网络q(s,a;w)q(s,a;w)q(s,a;w)。更新的过程如下：

算法相关

TD-Learning

核心思想就是认为下一步的估计更精确，用它作为target，以避免需要完成一次episode。

直接看Train DQN using TD Learning：

做了很多近似，得到：Q(st,at;w)≈E[Rt+γQ(St+1,At+1;w)]Q(s_t,a_t;w)≈E[R_t+\gamma Q(S_{t+1},A_{t+1};w)]Q(st​,at​;w)≈E[Rt​+γQ(St+1​,At+1​;w)]

所以设置TD Target：yt=rt+γQ(st+1,At+1;w)=rt+γ⋅maxa[Q(st+1,a;w)]y_t=r_t+\gamma Q(s_{t+1},A_{t+1};w)=r_t+\gamma· max_a[Q(s_{t+1},a;w)]yt​=rt​+γQ(st+1​,At+1​;w)=rt​+γ⋅maxa​[Q(st+1​,a;w)]，

就可以计算Loss了：Lt=12[Q(st,at;w)−yt]2L_t=\frac12[Q(s_t,a_t;w)-y_t]^2Lt​=21​[Q(st​,at​;w)−yt​]2，

梯度下降：wt+1=wt−α⋅∂Lt∂w∣w=wtw_{t+1}=w_t-\alpha·\frac{\partial L_t}{\partial w}|_{w=w_t}wt+1​=wt​−α⋅∂w∂Lt​​∣w=wt​​。

整个过程如下：

  1. 观测状态sts_tst​，做出动作ata_tat​；
  2. 用Q预测价值qtq_tqt​；
  3. 计算dt=∂qt∂w∣w=wtd_t=\frac{\partial q_t}{\partial w}|_{w=w_t}dt​=∂w∂qt​​∣w=wt​​。
  4. 环境返回新的状态st+1s_{t+1}st+1​和奖励rtr_trt​。
  5. 计算TD Target yty_tyt​。
  6. 梯度下降，更新参数：wt+1=wt−α⋅(qt−yt)⋅dtw_{t+1}=w_t-\alpha·(q_t-y_t)·d_twt+1​=wt​−α⋅(qt​−yt​)⋅dt​。

Sarsa算法

State-Action-Reward-State-Action，是TD-Learning的一种。

它的TD target为yt=rt+γ⋅q(st+1,at+1;w)y_t = r_t+\gamma·q(s_{t+1},a_{t+1};w)yt​=rt​+γ⋅q(st+1​,at+1​;w)

TD error为δt=q(st,at;w)−ytδ_t=q(s_t,a_t;w)-y_tδt​=q(st​,at​;w)−yt​。

上述Actor-Critic Methods中的TD部分就是使用Sarsa。

Q-Learing算法

最大的区别就是TD target为yt=rt+γ⋅maxa[q(st+1,a;w)]y_t = r_t+\gamma·max_a[q(s_{t+1},a;w)]yt​=rt​+γ⋅maxa​[q(st+1​,a;w)]，上述DQN的TD部分就是使用Q-Learing。

Multi-Step TD Target

其实就是多迭代几步，有Ut=∑i=0m−1γi⋅Rt+i+γm⋅Ut+mU_t=\sum_{i=0}^{m-1}\gamma^i·R_{t+i}+\gamma ^m·U_{t+m}Ut​=∑i=0m−1​γi⋅Rt+i​+γm⋅Ut+m​。

Experience Replay 经验回放

解决两个问题：

  1. 一条Transition用完后就丢掉了，但其实是可以继续用的，主要是因为st,ats_t, a_tst​,at​之后，rt,st+1r_t,s_{t+1}rt​,st+1​是确定的（或者至少抽样分布是确定的）。和网络无关。
  2. 相关性：相邻两条Transition有相关性，是有害的。

使用一个replay buffer，储存最近n条transitions，然后更新网络使用SGD(Stochastic gradient descent 随机梯度下降)的方法，随机在buffer里抽取一条来更新现在的网络，注意在不同时刻，相同的transition计算得到的TD error一般也是不同的，因为网络更新了。

采样方法可以让δδδ更大的以更大的概率被抽到，相应的学习率调小。例如：

pt∝∣δt∣+ϵp_t \propto |δ_t|+\epsilonpt​∝∣δt​∣+ϵ 或 pt∝1rank(t)p_t\propto \frac1{rank(t)}pt​∝rank(t)1​，其中rank为δ排名；学习率为α⋅(npt)−β\alpha·(np_t)^{-\beta}α⋅(npt​)−β。

Target Network & Double DQN

原始的DQN对动作价值的估计是高估（overestimate）的：

  1. The maximization：

    真正的action-values：x(a1),...,x(an)x(a_1),...,x(a_n)x(a1​),...,x(an​)

    DQN的估计是有噪声的：Q(s,a1;w),...,Q(s,an;w)Q(s,a_1;w),...,Q(s,a_n;w)Q(s,a1​;w),...,Q(s,an​;w)，

    当然是无偏的：meana(x(a))=meana(Q(s,a;w))mean_a(x(a))=mean_a(Q(s,a;w))meana​(x(a))=meana​(Q(s,a;w))

    但由于QLearning选取了最大的Q，有：maxa(x(a))≤E[maxa(Q(s,a;w))]max_a(x(a))≤E[max_a(Q(s,a;w))]maxa​(x(a))≤E[maxa​(Q(s,a;w))]

    ，也就是期望而言，一直是高估的。而且主要是高估是不均匀的，因为transition占比越高，高估越严重。

  2. Bootstrapping自举：

    大概是由于上述高估了，然后梯度更新是自举的，导致越来越严重。

Target Network

使用另一个结构相同的DQN：Q(s,a;w−)Q(s,a;w^-)Q(s,a;w−)，参数每隔一段时间更新，可以是：w−=ww^-=ww−=w或w−=τw+(1−τ)w−w^-=\tau w+(1-\tau)w^-w−=τw+(1−τ)w−。

然后在算TD target时，yt=rt+γ⋅maxa(Q(st+1,a;w−))y_t=r_t+\gamma·max_a(Q(s_{t+1},a;w^-))yt​=rt​+γ⋅maxa​(Q(st+1​,a;w−))，

TD error：δt=Q(st,at;w)−ytδ_t=Q(s_t,a_t;w)-y_tδt​=Q(st​,at​;w)−yt​。

可以部分解决自举。

Double DQN

和Target Network的区别是select时使用的是主DQN的网络，即：

a∗=argmaxaQ(st+1,a;w)a^*=argmax_aQ(s_{t+1},a;w)a∗=argmaxa​Q(st+1​,a;w)
